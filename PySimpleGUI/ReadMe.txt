Install Latest version of Python
Install PIP

Run install.bat

Add API Key in first line of config.txt
Add time interval in seconds in second line of config.txt

Create a file (e.g. mymessage.txt that contains text to be sent)

for commandline app execute run.bat
for GUI based app execut runGUI.bat

Visit https://www.pushbullet.com/#settings/account

Add that token to 1st line of config.txt
